# Automated gates runbook

## Purpose

Reduce human error by running **structural gate proof automatically**. Humans only perform steps machines cannot do (OAuth consent click, UI-003E visual judgment).

```text
Agent / CI  →  npm run gate:runtime002c -- --write-evidence
Optional    →  human OAuth click + marker file (one time)
Agent       →  npm run peer-review:ollama -- --slice runtime-002c --write
```

## Decision tokens

| Token | Who sets it | Meaning |
| --- | --- | --- |
| `RUNTIME_002C_AUTOMATED_STRUCTURAL_GATE_PASS` | Gate script | Unit tests + runtime checks + cargo test passed |
| `RUNTIME_002C_LIVE_OAUTH_CONSENT_RECORDED` | Gate script when marker exists | Owner completed Google OAuth once in Tauri |
| `RUNTIME_002C_GATE_PASS_READY_FOR_PEER_REVIEW` | Gate script when marker exists | Structural + live consent recorded |
| `RUNTIME_002C_PEER_REVIEW_PASS_READY_FOR_DOWNSTREAM` | Peer review receipt | After Ollama/agent peer review |
| `UI_003E_PASS_OWNER_VISUAL_PROOF_COMPLETE` | **Owner only** | Scaffold visual proof on `:4488` — not automated |
